# BugTracker Pro 🛡️

Профессиональная система отслеживания технических ошибок и управления задачами для команд разработчиков. Проект реализован в рамках учебного курса ТРПО.

## 👥 Команда проекта
* **Айгерим** (akabanegekko@kstu.kg) — Backend-разработчик
  - Проектирование архитектуры БД (8 таблиц).
  - Разработка серверной логики на языке Go.
  - Настройка REST API эндпоинтов.
* **Айдана** (atbekeshova@kstu.kg) — Frontend-разработчик
  - Разработка пользовательского интерфейса (UI/UX).
  - Верстка страниц на HTML/CSS.
  - Интеграция фронтенда с API через JavaScript Fetch.

## 🚀 Стек технологий
* **Backend:** Go (Golang) 1.21
* **Database:** SQLite 3 (реляционная БД)
* **Frontend:** Vanilla JavaScript, HTML5, CSS3

## 📋 Функциональные возможности
- **Дашборд:** Визуализация статистики открытых и закрытых багов.
- **Трекинг:** Создание, редактирование и удаление отчетов об ошибках.
- **Управление:** Назначение исполнителей на конкретные задачи.
- **Приоритизация:** Классификация ошибок по уровню критичности (Low, Medium, High, Critical).
